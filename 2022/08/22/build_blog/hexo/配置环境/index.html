配置环境

需要安装的软件

  • Node.js
  • Git

安装自己系统对应的版本

检查

打开cmd (Win + R, 并输入cmd, 然后Enter)

输入：

node -v

npm -v

git --version

如果出现对应程序的版本号就表明正确安装，否则请尝试重新安装或上网查询。

例如：

C:\Users>node -v
v16.16.0
C:\Users>npm -v
8.15.0
C:\Users>git --version
git version 2.37.1.windows.1

推荐安装的应用

  • PicGo

安装 Hexo

npm 一键安装 Hexo 博客程序：

npm install hexo-cli -g

p.s. Mac 需要管理员权限（sudo）:

sudo npm install hexo-cli -g

接着耐心等待就完成了。

创建GiitHub Page

创建库

GitHub 主页右上角加号 => New repository
Repository name中输入 用户名.github.io
Description (描述) 选填
Initialize this repository with a README可以勾选
点击 Create repository 创建

创建后默认博客地址为：https://用户名.github.io
